# User Stories

## Learner

As a learner, I want certification guidance so that I can prepare effectively.

As a learner, I want simple explanations of complex AI concepts so that I can build confidence.

As a learner, I want practice questions so that I can identify knowledge gaps.

## Trainer

As a trainer, I want session plans so that I can prepare learning delivery faster.

As a trainer, I want practical activities and assessment questions so that learners can apply concepts.

As a trainer, I want guidance on delivery risks so that I can improve learner engagement.

## Program Manager

As a program manager, I want suggested KPIs so that I can track program effectiveness.

As a program manager, I want structured stakeholder updates so that I can communicate progress clearly.

As a program manager, I want intervention recommendations so that I can address low attendance or completion.

## Administrator

As a learning administrator, I want reminder workflows so that certification and attendance follow-ups are consistent.

As a learning administrator, I want structured operational processes so that records remain accurate and complete.
