# AI Learning Operations Hub

## LearnFlow AI

LearnFlow AI is an AI-powered learning operations assistant designed to support learners, trainers, program managers, and learning administrators.

The project is inspired by real operational challenges in large-scale learning programs, including repetitive trainer queries, learner support, certification guidance, program reporting, content management, and stakeholder communication.

## Current Version

The current version includes:

- Role-based assistant experience
- Learner, Trainer, Program Manager, and Administrator roles
- Microsoft Foundry model integration
- OpenAI Responses API
- Conversation context tracking
- Streaming responses
- Role-specific AI instructions
- Responsible-use safeguards

## Business Problem

Enterprise learning teams spend significant time answering repeated questions, preparing learning plans, supporting trainers, tracking learner progress, and creating operational reports.

These activities are often manual, fragmented, and difficult to scale.

## Proposed Solution

LearnFlow AI provides role-based conversational support for:

- Learners preparing for certifications
- Trainers planning and delivering sessions
- Program managers monitoring learning outcomes
- Administrators managing learning operations
